📚 Quiz Application (JavaScript)

A simple and interactive Quiz Application built using HTML, CSS, Bootstrap, and JavaScript.
This project helps users test their knowledge with multiple-choice questions and a timer-based system.

🚀 Features
✅ Dynamic Questions & Options
⏱️ 30-Second Timer for each question
📊 Real-time Score Tracking
📄 Detailed Result Summary
🎯 Auto move to next question after selection
❌ Handles unanswered questions
💡 Clean UI using Bootstrap
🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5
JavaScript (Vanilla JS)

⚙️ How It Works
1. Load Question
Displays current question and options
Updates question number
2. Timer Function
Starts 30-second countdown
Automatically moves to next question if time ends
3. Answer Selection
User clicks an option
Answer is stored
Moves to next question instantly
4. Score Calculation
Correct answers increase score
5. Result Display
Shows:
Total Score
Each Question
Your Answer
Correct Answer
